「素通り禁止」について

Thu, Aug 25, 2005 in 未分類 using tags

諸事情により、タイトルを変更しています。

元タイトル: 最近のＮＥＴのギモン～その１・素通り禁止～

2011-01-26 rmn.

今日から、「最近のＮＥＴのギモン」と題しまして、
最近のインターネットで気になることを書いていきたいと思います。
 
ギモン０１「素通り禁止」
素通り禁止―それは楽天サイトでよく見かけられる光景・・・・・
意味は掲示板に必ず書き込んでください！という意味・・・・
「友達を増やしたい」そういう思いからそういうことをするらしい。
しかし、「素通り禁止なので書き込みました」とかきこむと
そういう書き方大嫌い！（だったかな？）と言われました・・・・
友達から「こんなサイトがあった」と報告を受けました・・・・
「素通りしたらプロバイダに通報します」・・・・・
もうわけがわかりません！そう書いたひとは冗談で書いているかもしれないが、
プロバイダにほんとうに通報したとしたら「はぁ？」と言われるはずです。
そんな内容が書かれているＷＥＢページはけなされて結局ヒット数が下がってしまうと思います。
あまり、素通り禁止はふさわしくない行為だと僕は思います。
 
以上で終了とします。
 
まつ部屋でもこれから素通り禁止サイトとならないように運営していきたいと思います。
 
次回の疑問では「目がチラチラするページ」をとりあげたいと思います。
 
次回もお楽しみに！
