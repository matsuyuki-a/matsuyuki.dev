４年目。。。

Thu, Jan 4, 2007  in 未分類 using tags

総アクセス数 32606アクセス　（約29アクセス/日）

開設日 2003年12月18日　（開設1114日）
 
これが、まつ部屋２００７の現状です。。。
僕がこのページここまでひっぱってきたのもみなさんのおかげです。
裏６メンバー（ハセコー１０７氏など）・新まつＮＥＴ（リク氏など）
・楽天で友達になってくれた人（ぽぷり氏など）ＴＯＷＮの人たち（まい氏など）・・・
さまざまな人たちの協力があってまつ部屋は３年生き残ってきました。
日記記入率は非常に少ないですが、今までに
北極マート・参加同盟集・ＷＷＡ・むにむに・ＴＯＷＮ・・・
といういろいろなコンテンツを運営してきました。
ＷＧＳという大規模サイトも運営してきました。しかし・・・
「閉鎖」という事態が現在まで６回もあったが故に、全面閉鎖という選択を迫られました。
 
Ｗｅｂサイトの運営ということは、大変頭を使う作業、あるいはその繰り返しです。
小学５年生のころから、中学受験による多忙期を経て、今ここにまつ部屋がまだ存在する、
そういうことだけでも嬉しいです。
 
２００３年２月に同級生の人（現在の瑠宇紀氏）がＷｅｂページをたて（２００４年度閉鎖）
２００３年１０月に自分が親の許可を得ずにインターネット上に姿をあらわした。
１２月１７日にPon-c.net（めがね委員長氏）と物知りハウス（ハセコー１０７氏）が出現。
（上記２人ともに同級生）
そして、同年１２月１８日にみんなと１日遅れで自分のページ（眞露講北極マート）をもちました。
しかし、次の年１０月には受験勉強によりまつ部屋初の閉鎖をしてしまいました。
（閉鎖と言っても、１ヶ月に１回は更新していました。）
しかし、１２月には親の反対を押し切って１周年記念祭をしました。
その頃、楽天広場ランダムからmaron0645氏（現在のぽぷり氏）が出現、
初の遠隔友好人物（？）となった。
来年２月には元の更新体制に戻りました。
そして、小学校を卒業後、中学校に入ってもまだ更新を続け、
中学で友達になった人たちにも自分のページや楽天広場によるブログの作り方なども説明しました。
（ここで言う友達はリク氏・ノイローゼ松氏などです。。。）
その後、ＷＷＡなどのゲームに着手するようになり、ここまでで７０００ヒット。
その後、ヒット数が非常に上がり、一気に２００００ヒット。
このころ、ゲーム専用ページ「まつひらゲームズ」も完成しました。
しかし、その後、テスト成績悪化のため、母者にパソコン禁止され３回目、４回目の閉鎖がありましたが
翌年３月にいよいよ「Web Game Station」が完成します。
さらに、「暇人の村」「ぶっとび」「ヒラリラーのＨＰ」などのサイトと相互リンク開始。
さらなる発展をとげました。（例としてＣＧＩゲーム「ＴＯＷＮ」の設置など）
しかし、相次ぐ閉鎖と荒らしの増加の影響でＷＧＳは閉鎖、まつ吉の部屋もまつ吉の小部屋となり、今に至ります。
 
と、ここまで、まつ吉の部屋の歴史を書き綴ってみた。まちがっていることもあるかもしれないが
確実にいままでの中で進歩はあった。そして、このような糞ページが３年間も続いたことを
正直に言って誇りに思っている。まだ改善するべき点はいっぱいあるなと思いつつ、この駄文を終える事とする。
平成１９年１月４日
まつ吉の小部屋管理人
まつ吉

前日の続きです。
 
２００５年１０月上旬。
世にも危険な旅行記公開。
　　まつ吉とヒラリラーの初合作ゲーム。
　　メールでdatファイルを交換しあって完成させるという新感覚でした。
　　もともとこのゲームはヒラリラーが最初に企画したゲームで、
　　当初は「脱北」というタイトルでした。
　　これはヒラリラーが最初に北チョンセン（≒北朝鮮）から脱出するゲーム
　　を私に送ってきたことから始まったためです。
　　そのため、今でも、通称は「脱北」のままです。
　　当時ではすごくタイムリーなゲームだったと思っています。
　　また、合作ではまた別のゲーム「遺跡探検記（または脱出記とも言うが・・・）
　　も製作予定でしたが、一瞬にして没落しました………
 
２００５年１０月２０日。
Ｖ.Ｓ.さいたま公開。
　　現在は公開終了しているゲーム。
　　理由はヒラリラーのＨＰで公開されているから。
　　もともとはヒラリラーがまつ吉に公開してくれ、
　　と要望があったので公開したもの。
　　しかし、それは未完成であった。
　　しばらくして、完成品が送られてきた。
　　そして、まつ吉が最後に少し加工したものだ。
　　しかし、ヒラリラーから「総監督」がまつ吉になっているという苦情がきた。
　　これに伴い、ヒラリラーのＨＰが出来た数日後、公開停止となってしまった。
　　チャンチャン♪
　　（追加事項：現在、ヒラＨＰで公開されているものはさらに加工されたV1.75です。）
 
２００５年１０月３１日。

　　これは（中学）受験時代以来のことであった。
　　理由はもちろん成績不良。
　　しばらくは地獄の日々が続いた…..
 
２００５年１２月１８日。
閉鎖状態のまま２周年突入。
　　当時予定していた記念祭は没になってしまった。。。。

　　自分がゲームを作っていないことを成績のせいではなく

　　という思いをこめて作ったゲーム。
　　でもはっきり言って没ゲームを強引につなぎ合わせただけなんだけどね・・・・・
 
２００５年１２月２３日。
「異常杉共和国」公開。
　　「１日で作ることが可能なクソゲー」をコンセプトに開発を続けたら
　　一瞬で終わる超クソゲーと化してしまった。
　　没ネタよりは楽しめるが掲示板がない……..
 
２００５年１２月２４日。

そういや、あの日から一年が経とうとしているな・・・

Fri, Jun 23, 2006  in 未分類 using tags

２００５年７月９日。
この日が何の日かご存知でしょうか。
知っている人は少ないでしょうがＷＷＡ「まつ吉迷宮」（初代）の公開日です。
よく考えると、あの日から僕のWebページは始まったようなものだな・・・・・。
でもWebページは２年半も前から始まっていたし・・・。
 
そう悩んでいるときりがないのでこっから先はまつ系ゲームの１年間を振り返ります。
 
２００５年６月。
隠しページっぽいゲーム「まつ吉迷宮」ができないか、と考える。
　　この時点ではまだＷＷＡのメイキングツールについて知りませんでした。
　　ＷＷＡそのものは知っていましたが・・・・・
 
２００５年６月下旬。
ＮＡＯ氏作成ゲーム「ケーブダンジョン」を改造できないか、と考える。
　　このゲームの改造で「まつ吉迷宮ＷＷＡスペシャル」が出来ないかと考えていました。
 
２００５年７月上旬。
「まつ吉迷宮」製作（といっても改造だけだが・・・・・・・・・・）開始。
 
２００５年７月９日。
初代作品「まつ吉迷宮ＷＷＡスペシャル」完成。
 
２００５年７月中旬。
ＷＷＡでまた別の作品ができないか、と試行錯誤。
　　このときすでに２つのゲームが没になってしまいました.....。
 
２００５年７月１７日。
２代目作品「まつ吉公園でのかくれんぼＷＷＡスペシャル」完成。
　　このゲームは当時の作品にしては完成度が高いです。
　　......といっても未解決のバグが２つあるし・・・・・・・
　　バグ１：女が木から何回も落ちる。
　　　　　　現在の技術では修正可能ですが、当時の感じを出すため、無修正となっております。
　　バグ２：ランキングＣＧＩ故障。
　　　　　　このゲームでは途中で改良が行われました。
　　　　　　第一回改良で、当初「まつ吉島」が「まつ吉公園」に。
　　　　　　第二回改良で、ランキングＣＧＩ、制作室設置。
　　　　　　この改良で発生したバグが今でも残っています。
　　　　　　具体的にどういうバグかというと、ランキングのスコアが０になるというものです。
　　まつ吉～。早く直せー！
 
２００５年７月２２日。
まつ吉の部屋の本館の移転作業が行われました。
　　これにより、現在のＷＧＳのサイトのところにまつ吉の部屋本館が現れました。
　　ちなみに本館は６月６日にできました。
　　本館の工事は「まつ吉の部屋」が「まつこっくりの部屋」だったころから続きました。
　　...といっても実際の作業時間＝３日・・・
　　詳しくは、次シリーズの日記あたりで公開すると思う「まつ吉の部屋　完全な歴史」をご覧下さい。
　　この日にはギロチンチャットの独立も行われました。
　　この理由はやはりＪｅｔＣｈａｔは重いということと、部活内でギロチンゲームがあまりにも
　　流行りすぎたことがあげられます。
　　本館もギロチャも忍者ツールズ（http://www.shinobi.jp/）が使われました。
　　ちなみにここからアクセス制限機とアクセス解析を借りています。
 
２００５年８月２日。
あの強烈にムズイゲーム「１本道」の完成した日。
　　実はあのゲーム、最初はイライラ棒のつもりで作ったのだが、
　　床落下システムやワープゾーンができたため崖っぷちの道を延々と歩き続けるゲームに・・・
　　途中からセーブが禁止になったので制覇者はいまだに１人・・・
　　注）作者はクリアしていません。
　　詫）あまりにも難しかったことをお詫び申し上げます...（・＿・）
 
２００５年８月１８日。
ゲームランキングの更新。
　　このころはゲームのランキングが存在しました。
Ｔｏｐ画像Ｆｌａｓｈ化。
　　Palafla!でFlashを作ってみる実験をしました。
　　結局ＡＳ学習不足により、まつ吉製Flashゲームはできませんでした。
　　これからできるかも・・・だけど・・・？
 
２００５年９月２４日。
ＷＷＡ追加。
　　資料がないため何が追加されたかは不明ですが以下の３つがこの時期に作成されました。
　　・ギ・ロウチンをたおすゲーム
　　・後戻り禁止
　　・没ネタＲＩＭＩＸ０１
　　「ギ・ロウチン」は当時の技術の集大成！
　　正男が混ざっていたのがよかったのかな・・・？
　　「後戻り禁止」は「The Big House」に入って脱出するストーリーがゼロのクソゲー。
　　エキスパートクリアがあるのだが、バグ未修正・・・・
　　「没ネタ」は全く楽しめないゲーム。
　　後で出てくる「異常杉共和国」の方が楽しめます。
　　０２もあるのだがサーバーにＵｐしたままずーーーーーーっと放置......
 
続く。次回もお楽しみに。
